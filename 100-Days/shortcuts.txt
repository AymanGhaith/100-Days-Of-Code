- code formatting: (option + shift + F)
- delete: (Fn + backspace)
- emmet: (type the tag name without angle brackets)
- spell check (command + .)
- move line (option + up-arrow)
- command palette (command + shift + p)
- open terminal: (control + shift + `)
- toggle block comment: (command + /)
- toggle line comment: ( shift + option + A)
- copy line above: (shift + option + arrow)
